# AmoCRM Copy Lead Widget

## Описание
Виджет для AmoCRM, который позволяет копировать текущую сделку с сохранением контактов и тегов.

## Использование
1. Открой карточку сделки.
2. Найди виджет "Скопировать сделку" на виджет-панели.
3. Выбери воронку и этап для новой сделки.
4. Нажми кнопку "Скопировать".
5. Новая сделка создастся с копией информации и привязанными контактами.
